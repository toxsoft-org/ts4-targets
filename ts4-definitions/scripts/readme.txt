В этой директории находятся скрипты и настройки (в основном, для ant), используемые для сборки
  общих библиотек и программ ТоксСофт.
   
  Ранее сборочные скритпы в каждом репозитории были свои. Сейчас в репозитории остается скрипт(ы) запуска сборки, которые
  фактически только перечисляют собираемые  артекфакты, а сама сборка происходит скриптами из этого директория.
